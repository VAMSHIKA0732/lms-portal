# LMS Portal - Platinum Edition

A full-stack Learning Management System (LMS) built with React, Node.js, and MongoDB.

## ✨ Features
- **Role-Based Dashboards**: Distinct experiences for Teachers and Students.
- **Course Management**: Teachers can create, manage, and monitor courses.
- **Assignment System**: Create assignments with due dates, submit tasks, and receive grades/feedback.
- **Real-time Discussion Forum**: Interactive course-specific messaging.
- **Material Sharing**: Upload and access resource links and documents.
- **Glassmorphic UI**: High-end, dark-themed design with smooth animations.

## 🛠️ Technology Stack
- **Frontend**: React, Vite, Lucide Icons, Axios.
- **Backend**: Node.js, Express, MongoDB, Mongoose.
- **Auth**: JWT-based secure authentication.
- **Styles**: Vanilla CSS for maximum performance and premium aesthetics.

## 🚀 Quick Start
### Prerequisites
- Node.js & MongoDB installed.

### Installation
1. Clone the repository.
2. Run `npm run install-all` from the root directory.

### Running Locally
- Start the server and client: `npm start` (Runs API and serves frontend build).
